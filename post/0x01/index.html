#更新

git add . 和 git add * 区别

git add . 会把本地所有untrack的文件都加入暂存区，并且会根据.gitignore做过滤，但是git add * 会忽略.gitignore把任何文件都加入 添加当前目录全部文件到暂存区

因此，当我们更新博客的时候，使用add * ，这样就可以保证上传的都是hugo命令生成的了。
